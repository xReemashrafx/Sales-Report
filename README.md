# Sales-Report
A simple Python program created as part of the "Getting Started with Python" course. 
Features:
- Input validation (sales must be between 1 and 5000).
- Calculates:
- Highest sale
- Lowest sale
- Average sale
- Number of big vs. small sales
- Percentage of big and small sales
- Handles invalid entries gracefully
  
Skills Demonstrated:
-Python dictionaries
-Loops and iteration
-Conditional statements
-Calculating percentages
-Basic statistics
